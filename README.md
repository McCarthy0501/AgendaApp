estructura inicial del proyecto AgendaApp

AgendaApp/
├── main.py              <-- El punto de entrada (el que ejecutas)
├── database.py          <-- Todo lo relacionado con SQLite (el motor)
├── gui.py               <-- El diseño de la ventana y botones (la cara)
├── logic.py             <-- Funciones de validación (el cerebro)
├── assets/              <-- Carpeta para iconos o imágenes (opcional)
└── data/                <-- Carpeta donde se guardará el archivo .db
